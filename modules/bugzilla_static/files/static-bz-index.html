This is the static, read-only HTML version of Wikimedia Bugzilla.

Bugzilla was replaced by Wikimedia Phabricator in November 2014. All bug reports were migrated to Phabricator. For up-to-date information, please visit the corresponding task in Wikimedia Phabricator instead.

You can find old bugs and historical activity in this static HTML version. Example links:

  • bug1.html
  • activity1.html

Replace '1' with your bug number.

Links in the original Bugzilla format such as show_bug.cgi?id=42 or show_activity.cgi?id=23 also work and are internal redirects to the HTML files.

So you can simply add "static-" to old Bugzilla URLs to get here.
