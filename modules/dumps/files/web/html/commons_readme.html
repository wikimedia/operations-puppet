Analytics Datasets: Commons Impact Metrics

This collection of datasets details how commons media is edited, used, and accessed across Wikimedia projects. Currently, we are publishing data about categories on an allow-list, curated jointly with the GLAM community.

The files available for download are all in TSV (tab-separated-value) format, with lists separated by the "|" character. Schemas for each dataset are detailed on-wiki: Commons_Impact_Metrics/Dumps. Lots more detail is also available on the parent page: Commons_Impact_Metrics.
