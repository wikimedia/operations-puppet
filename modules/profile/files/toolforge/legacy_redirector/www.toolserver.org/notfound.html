The Wikimedia Toolserver was a collaborative platform providing Unix hosting for various software tools written and used by Wikimedia editors and operated by Wikimedia Deutschland e.V. It has been replaced by Wikimedia Cloud Services, with the services that used to be provided by the Toolserver available through the Toolforge project.
