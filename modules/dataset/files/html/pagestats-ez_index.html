Notes about the format of the pagecount files

These are derived from Domas' pagecount files but the format is not identical. Each line contains four fields separated by spaces:

  • wiki code (subproject.project)
  • article title
  • monthly total (with interpolation when data is missing)
  • hourly counts
In the wiki code, the subproject is the language code (fr, el, ja, etc) and the project is one of b,k,n,q,s,v,z, corresponding to the projects below:
  • b:wikibooks
  • k:wiktionary
  • n:wikinews
  • q:wikiquote
  • s:wikisource
  • v:wikiversity
  • z:wikipedia
Hourly counts can be deciphered as follows:
Hour:
from 0 to 23, written as 0 = A, 1 = B ... 22 = W, 23 = X
Day:
from 1 to 31, written as 0 = A, 1 = B ... 25 = Y, 26 = Z, 27 = [, 28 = \, 29 = ], 30 = ^, 31 = _

Source for this information is http://lists.wikimedia.org/pipermail/wikitech-l/2011-August/054644.html.
