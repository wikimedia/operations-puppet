This is the static HTML version of Wikimedia Foundation Bugzilla.

You can find old bugs and activity here. Example links:

  • bug1.html
  • activity1.html

Replace '1' with your bug number.

Links in the original Bugzilla format such as show_bug.cgi?id=42 or show_activity.cgi?id=23 also work and are internal redirects to the HTML files.

So you can simply add "static-" to old Bugzilla URLs to get here.
