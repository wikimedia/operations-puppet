clear-profile(8)
==============
:doctype: manpage

NAME
----

clear-profile - delete graphite profiling data

SYNOPSIS
--------

*clear-profile*

DESCRIPTION
-----------

*clear-profile* will ping the graphite installation and ask it to remove the
profiling data.

OPTIONS
-------
This script has no option.

COMMANDS
--------
Describe commands, you can give some synopsis there such as:

  clear-profile

BUGS
----

This script does not handle any error while sending the ping back and will
always assume the command worked successfuly.
